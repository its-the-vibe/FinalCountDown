# FinalCountDown

[![CI](https://github.com/its-the-vibe/FinalCountDown/actions/workflows/ci.yaml/badge.svg)](https://github.com/its-the-vibe/FinalCountDown/actions/workflows/ci.yaml)

Webpage showing a count down for big events

## Development

### Makefile targets

| Target | Description |
|--------|-------------|
| `make build` | Compile the application |
| `make test` | Run all unit tests |
| `make lint` | Run `go vet` to check for common issues |
| `make ci` | Run build, test, and lint (all CI steps) |
